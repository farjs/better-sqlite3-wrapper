## @farjs/better-sqlite3-wrapper

Thin api wrapper around [better-sqlite3](https://github.com/WiseLibs/better-sqlite3) and [bun:sqlite](https://bun.sh/docs/api/sqlite) to allow cross- runtime/engine usage.
